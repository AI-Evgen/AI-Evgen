## 🐧 Parrot OS vs Kali Linux
- **Kali** — стандарт де-факто, огромная база инструментов.
- **Parrot** — легче, ориентирован на приватность, лучше для слабых ПК.

> **Совет:** Начните с Parrot OS, а когда освоитесь, при необходимости переходите на Kali.

---

## 📱 NetHunter: что можно делать?
- **Wi-Fi/Bluetooth-атаки** (проверка беспроводных сетей).
- **Быстрое сканирование** локальной сети.
- **Демонстрация навыков** «на ходу» (для презентаций).
- **Не заменит ПК** для веб-пентеста и сложных задач.

> ⚠️ **Важно:** SQL-инъекции, взлом веб-приложений и работа с большими объёмами данных **требуют ПК**. NetHunter — дополнительный инструмент, а не основная платформа.
